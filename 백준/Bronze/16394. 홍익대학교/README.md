### 입력 

 <p>입력으로 첫 줄에 특정 년도를 알리는 정수 N이 주어진다. 정수 N은 1,946 부터 1,000,000 사이의 값이다. (1,946 ≤ N ≤ 1,000,000)</p>

### 출력 

 <p>출력으로 홍익대학교의 개교 주년을 나타내는 정수를 출력한다.</p>
